# AndroidTVLauncher

![][image-1]

这是一个TV 桌面，包含了一个Leanback 桌面（最低版本是17）和一个普通桌面（最低版本是15），Leanback桌面是基于Leanback库开发符合Android TV官方交互规范。

### 注意
CatLauncher(Leanback风格) 目前功能已基本稳定，后续会跟进官方Leanback库的进展，Compatible桌面理论上不再维护

### 特性
> * Leanback风格
> * 图片展示
> * 视频直播（节操播放器-基于ijkplayer）
> * 媒体详情
> * 第三方应用列表
> * 桌面特殊功能
